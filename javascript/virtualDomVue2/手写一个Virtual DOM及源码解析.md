- `Virtual DOM`是当今主流框架普遍采用的**提高 web 页面性能**的方案，其原理是：
  - 1.把真实的 DOM 树转换成 js 对象（虚拟 DOM）
  - 2.数据更新时生成新的 js 对象（新的虚拟 DOM）
  - 3.二者比对后**仅对发生变化的数据进行更新**
